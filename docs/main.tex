\abstract{
We describe our hunter-gartherer system
for the NTCIR-10 1CLICK-2 task. 
We inspire ourselves on the DeepQA framework looking to adapt
it for the 1CLICK task.
Several techniques can be integrated naturally in this framework.
The hunter component generates candidates based on the passage retrieval for the original query, 
the gartherer component collects evidence for each candidate and score them based on the collected evidence, 
and finally a summarization system is utilized to organized the high-quality candidates.
The evaluation results show the effectiveness of this framework in the 1CLICK search task.
}
